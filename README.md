# MODIS Mosaic

## Installation

Install docker

docker-compose up -d

Open the website: localhost:8888

the password for the jupyter notebook is 123

## docker commands 

Docker pull ubuntu

Docker images

Docker ps -a

Docker run -it ubuntu bash

Docker rm <container id>

Run container:

docker exec -it <container id> bash
